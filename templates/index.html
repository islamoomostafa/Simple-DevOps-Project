Project Details

Introduction

In this project, I set up a complete development environment on AWS EC2. The project began with creating an Ubuntu instance, where I installed Jenkins, Docker, and Flask to develop a small Python application.

Steps Taken

1. Setting Up AWS EC2 Instance

Created an Ubuntu instance on AWS EC2 to serve as the base environment for developing the application.

2. Installing Jenkins, Docker, and Flask

Installed Jenkins, Docker, and Flask on the instance to enhance the development and deployment process.

3. Configuring Jenkins Pipeline

Created a Jenkins Pipeline to link the project to GitHub and execute build, install, and run steps for the application.

4. Managing Dependencies Without Docker

Started with a requirements file to manage dependencies for running the application without using Docker.

5. Enhancing the Project with Docker

Enhanced the project by using Docker to simplify environment management and standardization. Created a Dockerfile to build a Docker image for the application.

6. Pushing the Docker Image to Docker Hub

Built a Docker image of the project and pushed it to Docker Hub to make it available for future use.

Project Outcomes

Successfully set up a comprehensive development environment using AWS, Jenkins, Docker, and Flask, which improved project management and simplified deployment and future use.
