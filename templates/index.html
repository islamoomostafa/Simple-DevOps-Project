Welcome to My Simple DevOps Project

Project Overview

In this project, I created a small Python application using Flask, and deployed it using Docker and Jenkins on an AWS EC2 instance.

Setup Instructions

  • Created an Ubuntu instance on AWS EC2.
  • Installed Jenkins and Docker on the instance.
  • Configured a Jenkins pipeline and linked it to my GitHub project.
  • Developed a Flask application and created a Docker image for it.
  • Pushed the Docker image to Docker Hub for future use.
