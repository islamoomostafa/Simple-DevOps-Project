Welcome to My Simple DevOps Project

Project Overview

In this project, I started by setting up an AWS EC2 instance with Ubuntu. I then installed Jenkins and Docker on the instance to manage and build my project.

I created a small Python application using Flask and initially worked with a requirements file without Docker. Later, I enhanced the project by incorporating Docker, which allowed me to containerize the application.

Finally, I pushed the Docker image to Docker Hub to ensure that I could easily reuse the project in the future.

Project Steps

  • Created an Ubuntu instance on AWS EC2.
  • Installed Jenkins and Docker on the instance.
  • Configured a Jenkins pipeline and linked it to my GitHub project.
  • Developed a Flask application and created a Docker image for it.
  • Pushed the Docker image to Docker Hub for future use.
